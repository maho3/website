Latest Projects

Materials Search

A web-scraping tool for finding crystal structure data

Materials Search is a web-scraping and data-mining tool to aid researchers in finding new candidates for superconductivity. The tool searches crystal databases and paper records for information regarding the properties of possible crystal configurations. It processes this information using NLP and statistical analysis to provide useful data at a glance.

Materials Search is useful for consolidating information in order to draw inferences on a particular materialâ€™s magnetic and electronic properties. The tool runs on a Python flask server with a simple HTML front-end.
